## PromptX

一个简单的提词器。

> 别问为什么那么粉，问就是LD需要。

- [x] 悬浮题词
- [x] 字体大小、颜色、悬浮窗背景、题词速度
- [x] 自动识别复制文本
- [ ] 集成第三方朗读识别做文字跟随滚动
